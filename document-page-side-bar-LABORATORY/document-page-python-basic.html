Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

It was created by Guido van Rossum and first released in 1991.

Python's design philosophy emphasizes code readability and uses significant indentation to define code blocks.

Python is widely used in web development, data science, artificial intelligence, machine learning, and cybersecurity scripting.

It supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Most Linux systems like Ubuntu come with Python pre-installed. You can verify by running the command below in the terminal.

Python has a massive standard library and a large ecosystem of third-party packages available through PyPI.

Its syntax is close to plain English, which lowers the barrier of entry for new programmers.

Python is open-source and free to use on all major operating systems.

You can also run Python directly in the browser via platforms like Google Colab or Replit.

    • Variables and Data Types

    In Python, variables are containers for storing data values.

    Unlike other languages, Python has no command for declaring a variable — it is created the moment you first assign a value to it.

    Python is dynamically typed, meaning you do not need to specify the data type when declaring a variable.

    Python has several built-in data types: int, float, str, bool, and None.

    Strings can be enclosed in single quotes, double quotes, or triple quotes for multiline strings.

    Python supports f-strings (formatted string literals) introduced in Python 3.6 for easy string interpolation.

    The type() function returns the data type of any variable, which is useful for debugging.

    Python supports type casting using functions like int(), str(), and float().

    Variable names are case-sensitive and should follow snake_case convention.

    Variable names must start with a letter or underscore, never a number.

    # Variable assignment examples name = "John" # str age = 21 # int gpa = 1.75 # float is_enrolled = True # bool nothing = None # NoneType # Type casting age_str = str(age) # "21" pi = float("3.14159") # 3.14159 # F-string print(f"Hello, {name}! You are {age} years old.")
  • No need to declare variable types explicitly
  • Use snake_case for variable names (e.g., student_name)
  • Python is case-sensitive: myVar is not equal to myvar
  • None is Python's equivalent of null
  • Use type() to check the data type of a variable
    • Control Flow

    Control flow refers to the order in which individual statements and instructions are executed or evaluated.

    In Python, control flow is managed using conditionals and loops.

    The if-elif-else statement is the primary tool for decision-making in Python.

    Python does not have a traditional switch-case statement, though Python 3.10 introduced the match statement.

    Multiple conditions are chained using elif, which is short for "else if".

    Python provides two main loop types: the for loop and the while loop.

    The range() function is frequently used with for loops to generate a sequence of numbers.

    Python's indentation is critical — the body of an if block or loop must be consistently indented (typically 4 spaces).

    Using break exits a loop entirely, while continue skips the current iteration and moves to the next one.

    Both for and while loops support an optional else clause that runs when the loop finishes normally.

    # If-elif-else example grade = 88 if grade >= 90: print("Excellent!") elif grade >= 75: print("Passed!") else: print("Failed. Try again.") # For loop with range for i in range(1, 6): print(f"Count: {i}") # While loop count = 0 while count < 3: print(f"Running... {count}") count += 1
  • Python uses indentation (not braces) to define blocks
  • elif is used instead of else if
  • range(start, stop, step) generates number sequences
  • break exits a loop; continue skips the current iteration
  • for loops can iterate over strings, lists, tuples, and more
    • Functions

    A function is a reusable block of code that performs a specific task.

    Functions help you avoid repeating code, make programs easier to read, and break complex problems into smaller pieces.

    In Python, functions are defined using the def keyword, followed by the function name, parentheses, and a colon.

    Functions can accept parameters (inputs) and return values using the return statement.

    If no return statement is used, the function returns None by default.

    Python supports default parameter values, keyword arguments, *args, and **kwargs.

    Lambda functions are short, one-line anonymous functions written using the lambda keyword.

    Python also supports recursion — a function calling itself — useful for problems like computing factorials.

    Recursive functions require a base case to prevent infinite loops.

    Docstrings are triple-quoted strings placed right after the function definition to document what it does.

    # Basic function def greet(name, greeting="Hello"): """Returns a greeting message.""" return f"{greeting}, {name}!" print(greet("Ver")) # Hello, Ver! print(greet("Ver", "Kamusta")) # Kamusta, Ver! # Lambda function square = lambda x: x ** 2 print(square(5)) # 25 # Recursive function def factorial(n): if n == 0: return 1 return n * factorial(n - 1) print(factorial(5)) # 120
  • Define functions with the def keyword
  • Parameters can have default values
  • Use *args for variable positional arguments
  • Use **kwargs for variable keyword arguments
  • Lambda functions are for short, single-expression logic
    • Lists and Dictionaries

    Lists and dictionaries are two of the most commonly used data structures in Python.

    A list is an ordered, mutable collection of items enclosed in square brackets.

    Lists can contain items of different data types and allow duplicate values.

    Common list methods include append(), remove(), pop(), sort(), and len().

    A dictionary is a collection of key-value pairs enclosed in curly braces.

    Dictionaries are mutable and do not allow duplicate keys.

    You can access dictionary values using their keys and add new pairs by direct assignment.

    As of Python 3.7+, dictionaries maintain insertion order.

    List comprehensions provide a concise way to create lists based on existing iterables.

    Dictionaries are commonly used to handle JSON data, API responses, and database query results.

    # List example students = ["Ver", "Luja", "Jean", "Tadeo", "Ryan"] students.append("Ian Carlo") students.sort() print(students[0]) # Ian Carlo (after sort) print(len(students)) # 6 # List comprehension squares = [x**2 for x in range(1, 6)] print(squares) # [1, 4, 9, 16, 25] # Dictionary example student = { "name": "John Denver", "year": 3, "course": "BSIT", "gpa": 1.75 } print(student["name"]) # John Denver student["school"] = "CCSFP"
  • Lists use square brackets [ ]; dictionaries use curly braces { }
  • List items are accessed by index (0-based)
  • Dictionary values are accessed by their key
  • Both support comprehensions for concise code
  • Dictionaries are perfect for handling JSON and API data
